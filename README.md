# Pill Reminder

Минималистичное Android-приложение для графика приёма таблеток.

## Возможности

- Список своих таблеток — вводишь название один раз, дальше выбираешь быстрым доступом
- Напоминания со звуком на конкретное время:
  - 10 встроенных системных звуков на выбор
  - свой звук с телефона (любой аудиофайл)
- Цикличность напоминаний:
  - каждый день
  - каждые N дней
  - по дням недели
  - курсом N дней (автоматически завершается)
- Кнопка «Принял» прямо в уведомлении — запись попадает в журнал
- Календарь: какие таблетки и во сколько были выпиты, по дням
- Статистика: всего приёмов, за 7/30 дней, по каждой таблетке + среднее время приёма
- Напоминания переживают перезагрузку телефона

## Скачать APK

Готовая APK публикуется в [Releases](https://github.com/kipingame-cell/pill-reminder/releases) после каждой сборки.

## Сборка

GitHub Actions собирает APK автоматически при каждом пуше в `main` (`.github/workflows/build.yml`).
Локально: `gradle assembleDebug`.
